# THYAO RSI and CBRT Interest Rate Decisions

This project visualizes Turkish Airlines (THYAO) stock price and its 
14-day RSI alongside CBRT interest rate decisions between 2022-2026.

## Findings
The CBRT's shift toward orthodox monetary policy (rate hikes, shown in green)
coincides with THYAO's strongest upward momentum. This is a temporal overlap,
not evidence of causation.
One possible interpretation: both may reflect a broader period of
macroeconomic stabilization. A more predictable lira and improved investor
sentiment could support equity prices generally — and THYAO, with its
dollar-denominated revenues, may benefit particularly. However, this chart
alone cannot establish these mechanisms; they would require further analysis.

## Data Sources
- CBRT interest rate data: EVDS API
- THYAO stock price: Yahoo Finance API

## Libraries
pandas, matplotlib, evds, yfinance, python-dotenv
